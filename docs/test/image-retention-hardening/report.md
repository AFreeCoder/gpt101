### 行为覆盖

- 当前镜像和最近两个不同成功版本得到保留。
- 更旧成功版本和未成功上线的本地 `sha-*` 镜像得到清理。
- 同一镜像的多个标签不会重复占用保留名额。
- `--dry-run` 不调用镜像删除。
- 被容器引用的旧镜像不会被强制删除，并返回失败。
- 清理调用位于全部健康检查之后。
- 提交号回滚使用 `sha-<完整提交号>`。
