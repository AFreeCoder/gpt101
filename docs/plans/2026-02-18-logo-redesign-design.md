# GPT101 Logo 重新设计

## 日期
2026-02-18

## 背景
GPT101 从 Astro 迁移到 ShipAny (Next.js 16) 模板后，logo 仍为 ShipAny 的小船图标，需要替换为项目专属 logo。

## 设计需求
- **风格**：全新设计，不沿用旧站
- **形式**：图标 + 文字组合
- **配色**：蓝绿科技感
- **概念**：AI / 对话

## 选定方案：对话气泡渐变方块

### 图标
- 外形：圆角正方形（rx=22%）
- 背景：线性渐变 `#0EA5E9`（天蓝）→ `#10B981`（翠绿），左上到右下
- 内容：白色简约对话气泡 + 3 个圆点（AI 思考中）+ 气泡尾巴
- 留白：图标内容占 60%，四周保留呼吸空间

### 文字（完整 logo）
- 字体：无衬线粗体（Helvetica/Arial Bold）
- 排列：图标右侧，垂直居中
- 颜色：深灰 `#1E293B`，暗色模式白色

### 输出文件
| 文件 | 用途 | 尺寸 |
|------|------|------|
| `logo.svg` | 矢量源文件 | viewBox 0 0 512 512 |
| `logo.png` | 网站 header、OG 图片 | 512x512 |
| `favicon.ico` | 浏览器标签页图标 | 16/32/48 多尺寸 |

### 实现方式
1. 手写 SVG 代码
2. Python（Pillow + cairosvg）转 PNG 和 ICO
3. 替换 `apps/web-next/public/` 下的现有文件
